# ============================================================================
# PLATFORM-SPECIFIC NOTES
# ============================================================================

# Windows:
# - dlib-bin provides pre-built binaries to avoid compilation
# - pybluez2 may have limited support; consider Windows Bluetooth APIs
# - Some packages may require Visual C++ Build Tools
# - Run PowerShell script for smoothest installation

# Linux:
# - Install system dependencies: sudo apt-get install python3-dev libopencv-dev
# - pybluez2 should work natively
# - dlib may require: sudo apt-get install build-essential cmake
# - Consider using conda for easier dependency management

# macOS:
# - Install Xcode Command Line Tools: xcode-select --install
# - pybluez2 may have limited support
# - dlib may require: brew install cmake
# - Consider using conda for better compatibility

# pip install torch torchvision --index-url https://download.pytorch.org/whl/rocm5.7

# ============================================================================
# TROUBLESHOOTING COMMON ISSUES
# ============================================================================

# Issue: "dlib installation failed"
# Solution: Use dlib-bin on Windows, or conda install -c conda-forge dlib

# Issue: "pybluez2 not available on Windows"
# Solution: Skip Bluetooth features or use Windows-specific Bluetooth APIs

# Issue: "ultralytics too slow"
# Solution: Use YOLO_CONTEXT_MOCK=1 or install GPU-enabled PyTorch

# Issue: "mediapipe not found"
# Solution: Ensure Python 3.8+ and reinstall: pip install --upgrade mediapipe

# Issue: "opencv-python import error"
# Solution: Try opencv-python-headless for server environments

# Issue: "face_recognition import error"
# Solution: Install with --no-deps after other dependencies (see header)
