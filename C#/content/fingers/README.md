# Finger Count Images

Place PNG images named `0.png` through `5.png` in this folder.
Each image represents the number of fingers up detected by the mediapipe hand tracker.

| File   | Meaning          |
|--------|------------------|
| 0.png  | Fist / no fingers |
| 1.png  | 1 finger up       |
| 2.png  | 2 fingers up      |
| 3.png  | 3 fingers up      |
| 4.png  | 4 fingers up      |
| 5.png  | Open hand         |

Recommended size: 256x256 px, transparent background (PNG).

If images are missing, the app falls back to a drawn circle with the count number.
